    ElixirOC

    To view the docs open 

    doc/index.html

    Running locally:

    mix deps.get
touch config/dev.exs

    In your config/dev.exs:

    use Mix.Config

config :elixirOC, appID: "<appID>"
config :elixirOC, apikey: "<apikey>"
